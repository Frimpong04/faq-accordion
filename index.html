What is Frontend Mentor, and how can it help me?

Frontend Mentor offers realistic coding challenges to help developers improve their frontend coding skills with projects in HTML, CSS, and JavaScript. It's suitable for all levels and ideal for portfolio building.

How can I get help if I ma stuck on a project?

Frontend Mentor has a helpful community of developer just like you and you will get plenty of help there. Expect to use resources like google and stack overflow

Is Frontend Mentor free?

Frontend Mentor is free, but there is a premium version that is paid for. Thee free version is adequate as a beginner

Can I use Frontend Mentor projects in my portfolio?

Definitely, you can use Frontend Mentor's project as part of your portfolio of projects done
